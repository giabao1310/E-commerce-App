# E-Commerce App

This is a guide for using the web deployed

## Running json server

~~~
npm run start-json
~~~

With the fomart `start-json` can be changed in file package.json

## Login with 2 role
- admin:
  * username: `admin`
  * password: `admin`
- user:
  * usrename: `user`
  * password: `user`
 
Then you can review and check all the function that this web is having.
